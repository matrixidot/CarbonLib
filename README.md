# CarbonLib
A library to make plugin development easier

## How to use
Maven:

	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>


	<dependency>
	    <groupId>com.github.matrixidot</groupId>
	    <artifactId>CarbonLib</artifactId>
	    <version>v1.0.0-Beta</version>
	</dependency>
